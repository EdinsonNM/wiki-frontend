# Project Context

Sigue `AGENTS.md` y `GEMINI.md`.

No leas toda la wiki. Lee `wiki/index.md`, clasifica la tarea y usa la ruta de lectura minima.

No ejecutes comandos destructivos. No leas secretos ni artefactos generados salvo necesidad explicita.

Para tareas grandes:

1. Crear o revisar spec.
2. Planificar ownership.
3. Implementar cambios pequenos.
4. Ejecutar validacion.
5. Documentar resultados reales.

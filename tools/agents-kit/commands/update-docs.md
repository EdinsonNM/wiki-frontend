# Update Docs

Actualiza documentacion viva segun la implementacion real.

Lee:

- `AGENTS.md`
- `docs/README.md`
- archivos modificados
- comandos reales del proyecto

No inventes comandos. Si no pudiste verificar algo, marcalo como pendiente.

Actualizar cuando cambie:

- comportamiento visible
- arquitectura o boundaries
- comandos/scripts
- variables de entorno
- integraciones
- decisiones locales

No actualizar por:

- cambios internos sin efecto observable
- tests adicionales sin cambio de comportamiento
- formateo local

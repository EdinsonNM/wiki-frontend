# Frontend Testing

## Read First

- `AGENTS.md`
- `wiki/delivery/testing-and-coverage.md`
- `docs/quality/testing-strategy.md`
- `package.json`, si existe
- spec relacionada, si existe

## Workflow

1. Detecta framework de test.
2. Identifica comportamiento critico.
3. Agrega o ajusta tests proporcionales.
4. Ejecuta validacion relevante.
5. Reporta comandos y resultado.

## Do Not

- No bajes thresholds para hacer pasar tests.
- No cambies comportamiento productivo salvo que la tarea lo pida.

# Tokens

## Fuente De Verdad

Registrar donde viven colores, spacing, tipografia, radios, sombras y breakpoints.

| Token | Ubicacion | Notas |
| --- | --- | --- |
| Color | `src/design-system/tokens` | Ajustar si el proyecto usa otra ruta |
| Spacing | `src/design-system/tokens` | Ajustar si el proyecto usa otra ruta |
| Typography | `src/design-system/tokens` | Ajustar si el proyecto usa otra ruta |
| Radius | `src/design-system/tokens` | Ajustar si el proyecto usa otra ruta |

## Reglas

- No crear valores visuales arbitrarios si existe token equivalente.
- Si aparece un valor repetido en varias pantallas, promoverlo a token.
- No cambiar tokens globales durante una feature sin revisar impacto visual.

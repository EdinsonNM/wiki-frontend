# Integration Diagrams

Guardar diagramas de secuencia entre UI, estado, adapters, API y servicios externos.

## Usos Recomendados

- Secuencia de autenticacion.
- Fetch/mutation con loading, error y success.
- Flujo de refresh token o expiracion de sesion.
- Contrato entre frontend y API.

## Regla

Todo diagrama de integracion debe indicar fuente de datos, boundary, errores esperados y ubicacion del client/adapter.

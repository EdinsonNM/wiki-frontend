# Feature Diagrams

Guardar diagramas del flujo, estados o secuencias de esta feature.

## Tipos Utiles

- `flow`: pasos del usuario.
- `state`: loading, empty, error, success y transiciones.
- `sequence`: UI, hooks/services, adapters y API.
- `swimlane`: responsabilidades por actor o capa.

## Regla

Enlazar el diagrama desde `overview.md`, `user-flows.md` o `states.md`. No dejar diagramas sueltos sin contexto.

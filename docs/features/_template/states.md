# States

## Estados UI

| Estado | Descripcion | Fuente |
| --- | --- | --- |
| idle | Estado inicial | UI |
| loading | Esperando datos o accion | data fetching |
| empty | Sin resultados | API/dominio |
| error | Error recuperable | API/validacion |
| success | Operacion completada | UI/dominio |
